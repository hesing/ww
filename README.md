# Modular App

Install following

- Ruby
- Node JS

## Devlopment Setup

```
npm install
gulp or npm start
http://localhost:8080/dist/screens/
http://localhost:8080/dist/screens/ProjectDetail.html#
```

## Production Demo

```
node server.js
http://localhost:5000/screens/ProjectDetail.html
```
